# Tên dự án: Landing Page - Thắng Lợi Group (Bất động sản/Nội thất)

* **Mô tả:** Thiết kế và phát triển giao diện trang chủ cho doanh nghiệp kinh doanh bất động sản và nội thất. Trang web tập trung vào việc hiển thị hình ảnh không gian kiến trúc và phân loại danh mục dịch vụ rõ ràng.
* **Công nghệ sử dụng:** HTML5, CSS3 (Flexbox/Grid).

### Tính năng nổi bật:
* **Thanh điều hướng (Navigation bar):** Tối giản, chuyên nghiệp.
* **Thiết kế Banner tràn viền (Hero Image):** Tạo cảm giác không gian rộng.
* **Khu vực dịch vụ:** (Văn phòng, Biệt thự...) Sử dụng hệ màu tương phản (Vàng - Đen) để tạo điểm nhấn mạnh mẽ.
* **Footer:** Phân bổ thông tin liên hệ và chính sách theo dạng cột chuẩn UI/UX.

### Mục tiêu bài tập:
Thực hành kỹ năng bố cục layout, quản lý màu sắc và xử lý hình ảnh trong lập trình Front-end.
